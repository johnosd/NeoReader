# Android Log QA Checklist

Dispositivo: celular real conectado via Android Studio/ADB
Pacote: `com.johnny.neoreader`
Data da rodada: 2026-06-14

Use este arquivo para testar um fluxo por vez. Para cada item, iniciar uma captura limpa de logcat, executar somente o fluxo descrito, salvar o log e anotar o resultado.

## Como registrar cada teste

1. Iniciar captura com logcat limpo.
2. Executar apenas o fluxo do teste atual.
3. Parar a captura ou aguardar o tempo definido.
4. Registrar o caminho do log filtrado e do log completo.
5. Marcar status: `Pendente`, `Passou`, `Falhou` ou `Investigar`.
